# FOTUR VPN

FOTUR VPN — сервис защищённого подключения с быстрым входом, единым тарифом и аварийным временным доступом, если Telegram временно недоступен.

## Что такое FOTUR

FOTUR создан как простой и понятный способ получить приватный доступ без лишних экранов, кабинетов и перегруженных сценариев. Пользователь сразу видит, сколько стоит подписка, куда нажимать для покупки и как получить временную ссылку, если основной канал сейчас не открывается.

## Тариф

- `162 ₽ / месяц`
- `120 ГБ` трафика
- неограниченное количество устройств

## Быстрые переходы

- Купить подписку: [Telegram-бот FOTUR](https://t.me/foturvpnbot)
- Главная страница: [index.html](./index.html)
- Тариф: [tariffs.html](./tariffs.html)
- Загрузки: [download.html](./download.html)

## Если Telegram недоступен

На сайте предусмотрен аварийный сценарий: пользователь оставляет почту и получает временный доступ без ожидания. Это помогает вам беспрепятсвенно зайти в телеграм и купить полноценный доступ.

## Состав проекта

- `index.html` — главный экран, оффер, возможности и временный доступ
- `tariffs.html` — единый тариф и повторная точка входа в покупку
- `download.html` — загрузки и инструкции
- `features.html` — редирект на главную
- `style.css` — общий визуальный стиль
- `script.js` — интерфейсное поведение и логика форм
